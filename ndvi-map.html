Global NDVI Map

Earth Engine JavaScript Data Viz NDVI Analysis

This interactive map visualizes global vegetation health using the Normalized Difference Vegetation Index (NDVI). Built with Google Earth Engine, it utilizes the FAO Global Administrative Unit Layers Database for location selection, allowing users to explore NDVI trends for cities worldwide from 2010 to 2020.

NDVI values range from -1 to 1, where higher values indicate healthier, denser vegetation. The time series analysis reveals patterns of urban greening, seasonal variations, and long-term vegetation changes across different biomes.
